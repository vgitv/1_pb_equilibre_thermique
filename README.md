# Modèle de dérive diffusion stationnaire à l'équilibre thermique

Le but de ce TP est de résoudre l'équation de une dimension suivante :

$$ \partial_{xx} \Psi(x) = \delta_n e^{\Psi(x)} - \delta_p e^{-\Psi(x)} - C(x) $$
$$ n(x) = \delta_n e^{\Psi(x)} $$
$$ p(x) = \delta_p e^{-\Psi(x)} $$
$$ \Psi(x = 0) = \Psi^{D}_{l} $$
$$ \Psi(x = 1) = \Psi^{D}_{r} $$
